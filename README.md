백엔드 서버


## 🔧 Stack
- Nest.js, mysql


## 🔧 데이터베이스 구조

## 구현 기능
- 웹소켓
- 현 위치 주변 장소 검색
- 소셜 로그인(passport)
- 텍스트 기반 이미지 생성
- 운동 루틴 추천
- 키워드로 관련 정보 불러오기
- 비밀번호 찾기(메일 발송)
